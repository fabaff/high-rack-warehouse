Zustände
========

Beladen
-------
1 RBG auf 0/0 leer
2 RBG auf 0/0 beladen
3 RBG auf Y/Z beladen
4 RBG auf X beladen
5 RBG auf X leer
6 RBG auf Y/Z leer

Entladen
--------

1 RBG auf 0/0 leer
6 RBG auf Y/Z leer
5 RBG auf X leer
4 RBG auf X beladen
3 RBG auf Y/Z beladen
2 RBG auf 0/0 beladen

Nachfolgendens Entladen
------------------------
7 RBG auf Y1/Z1 leer
-> dann in normalen Entlade-Zuklus

--------

5 RBG auf X leer
4 RBG auf X beladen
3 RBG auf Y/Z beladen

3 RBG auf Y/Z beladen
4 RBG auf X beladen
5 RBG auf X leer
